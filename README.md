# Tools Gallery

A minimalist collection of essential web utilities for developers and designers. Fast, private, and easy to use.

## Features

- **Minimalist Design**: Clean, distraction-free interface with Inter typography.
- **Dark Mode**: Fully supported system-aware dark mode.
- **Tools Collection**:
  - **JSON Formatter**: Clean, format, and validate JSON data.
  - **Markdown Previewer**: Real-time markdown rendering.
  - **Base64 Converter**: Encode and decode strings/files.

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Start the development server**

   ```bash
   npm run dev
   ```

3. **Build for production**

   ```bash
   npm run build
   ```
